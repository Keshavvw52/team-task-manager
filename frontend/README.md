# Frontend

React client for Team Task Manager.

Most setup notes live in the root `README.md`. The short version:

```bash
npm install
copy .env.example .env
npm run dev
```

Set `VITE_API_URL` in `.env` if your backend is not running on `http://localhost:8000`.

Useful scripts:

```bash
npm run dev
npm run build
npm run lint
```
